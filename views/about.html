What is Leech?

Leach is an under-engineered link shortener. Leech has no database or server and can be configured with your own custom domain name. The only requirement is this 2 page application which you can launch for free on Heroku by clicking the button below.

Leech hooks up to a Google Analytics account which serves are your database and UI for reporting. This is also free.

How does Leech do this without a server? This application (the one that generates the URL) sets up a static website on AWS S3. This will run you about $0.01 a month.

Each short URL becomes a HTML page. Each page is about 844 Kb and when opened, it will trigger the Google Analytics API call as and perform the redirect to the full URL.
